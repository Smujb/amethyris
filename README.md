# Amethyris (Amethyst Iris) - Arch Linux bootc with swaywm

This is an opinionated bootc image based off of [bootcrew's Arch](https://github.com/bootcrew/mono). Currently intended only for personal use, I may adapt it into a more general purpose lightweight window manager setup down the line if I feel like it. For now, do not suggest features/additions and if you like my setup you would probably be better off forking it than making your own image `FROM ghcr.io/smujb/amethyris` as I may change things at any time.

Used alongside [my dotfiles](https://github.com/Smujb/dotfiles).
